# Turn Path Code Generation Reasoning


| `code element` |            |                                     |
| -------------- | ---------- | ----------------------------------- |
| `id`           | `E(refID)` | `the emotional state of the choice` |
| `id`           | `P(refID)` | `the physical action of the path`   |
| `id`           | `C(refID)` | `the cognitive action of the path`  |
| `class`        | `M(class)` | `the moral balance of the path`     |
| `level`        | `S(level)` | `The narrative stakes in the path`  |
