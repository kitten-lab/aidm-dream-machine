## **ven.2D-poly-quad**

Code

	ven.2D-poly-quad{
		$ 4 .
		Z :N -_
	}

Illustration
```
	*-*    *-*      *---*    *-----*    *
	| |     \ \     |  /      \   /    / \
	*-*      *-*    *-*        *-*    *   *
	                                   \ /
	                                    *
```

Use with:

|constrained rule added|resulting shape|
|---|---|
|equal edges|rhombus|
|right angles|rectangle|
|equal edges + right angles|square|
|skewed edges|parallelogram|
|3D lift|tetra frame|
